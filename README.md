# Sentry WSGI with Gunicorn/Gevent

This is a minimal test case for running the [Python SDK for Sentry](https://sentry.io/for/python/) with [Gunicorn](https://gunicorn.org/) and [Gevent](http://www.gevent.org/) workers.
